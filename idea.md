# Освітня платформа для інтерактивних панелей Promethean

## Концепція
Платформа для освіти на базі електронних панелей Promethean з інтерактивними завданнями на основі підходів **PhET** та **learning.ua**.

## Референси
- https://arbook.info/market/physics/vimiryuvannya-masi-til-za-dopomogoyu-vazhilnih-tereziv-ta-elektronnih-vag-ar-3d/
- PhET Interactive Simulations
- learning.ua

## Формат
- Браузерна платформа (працює без установки)
- Інтерактивна взаємодія через **дотик** (Promethean), **мишу**, інші засоби введення
- Контент генерується / реагує на взаємодію користувача

## Приклад: Лабораторна робота "Вимірювання маси тіл"

### Візуалізація
3D-сцена з робочим столом, на якому розміщені:

**Вимірювальні прилади:**
1. Динамометр (тензометр)
2. Механічні важільні терези
3. Електронні ваги

**Допоміжне обладнання:**
- Набір гирьок: 1 кг, 500 г, 100 г, ... (для важільних терезів)

**Об'єкти для вимірювання:**
- Тенісний м'яч
- Яблуко
- Бейсбольний м'яч

### Завдання
Учень повинен різними методами визначити масу запропонованих тіл, використовуючи всі три типи приладів, та порівняти результати.

## Що треба пропрацювати на брейнстормі
- Технологічний стек (Three.js / Babylon.js / React Three Fiber?)
- Архітектура контенту: як швидко додавати нові лабораторні
- Фізичний рушій (cannon-es / rapier?) для реалістичної ваги/гравітації
- UX для тач-панелі Promethean (великі цілі, мульти-тач, без hover)
- Модель контенту: scene description / declarative DSL?
- Як будуть оцінюватися результати учня
- Інтеграція з шкільними системами (LMS, журнал)
- Локалізація (укр / англ)
